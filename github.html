About Me

GitHub is a web-based hosting service for software development projects that use the Git revision control system.

GitHub offers both paid plans for private repositories, and free accounts for open source projects.

As of May 2011, GitHub was the most popular open source code repository site.

GitHub Inc. was founded in 2008 and is based in San Francisco, California.
